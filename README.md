## To Run a Local Copy

Clone the repo
Run `npm install` to install the dependencies

-   Create a `.env` file in the config directory and add/edit the following as `key: value`
    -   PORT: 2121 (can be any port example: 3000)
    -   DB_STRING: `your mongoDB URI`
